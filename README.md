# Healty.IO
Tic tac toe game

Instructions:
1. To start a new game, insert the following URL params:
 * start : tic tac toe
 * player1 : player name
 * player2 : player name
2. To play, insert the following URL params:
  * play : coordinate
  (coordinate format :  xy)

This project was coded in https://webtask.io/.
